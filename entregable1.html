    ENTREGABLES ENTREGABLE 2

    Creación de un repositorio de GitHub

    Para la creación de la página se utilizó la cuenta de GitHub de un integrante, en la cual se creó un repositorio para el proyecto. Posteriormente, se modificó las configuraciones de GitHub Pages; para publicar el sitio web, y Manage Access; para compartir el repositorio con los demás integrantes


    Elaboración del esquema principal de la página

    Para la elaboración del esquema principal de la página se utilizó el editor de código Visual Studio Code. En la página se utilizó lenguaje HTML, CSS, JavaScript y Jquery. Las fuentes utilizadas en la página se exportaron de Google Fonts y los íconos de Fontawesome. Cuando se terminó el esquema principal de la página se subieron los códigos al repositorio de GitHub del proyecto.


    Adaptación de la página a dispositivos móviles

    Para permitir la visualización de la página web en dispositivos móviles se utilizó los media queries de CSS para reajustar los tamaños de letra en función al tamaño del dispositivo. Asimismo se habilitó un menú con ícono para pantallas pequeñas.
